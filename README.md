# Dashboard-de-An-lise
Dashboard que eu desenvolvi, utilizando HTML, CSS e Javascript. Onde suas funcionalidades são:
✅ Página de Login com animação
✅ Dashboard principal com cards estatísticos
✅ Gráficos dinâmicos (usando Chart.js)
✅ Tabela de dados com filtros
✅ Perfil do usuário editável
✅ Configurações com toggle de tema
Obs: Esse projeto tem poucas funcionalidades pois seu maior foco é no seu visual.
